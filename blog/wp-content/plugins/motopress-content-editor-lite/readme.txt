== Description ==

MotoPress Content Editor enhances the standard WordPress editor and enables you to build websites visually. It's a complete solution for creating responsive pages without coding - add content elements in clicks and move them around the page with drag and drop. MotoPress [Page Builder](https://motopress.com/products/content-editor/?utm_source=wp-org&utm_medium=textlink&utm_campaign=content-editor-lite&utm_content=motopress-page-builder) plugin is easily extendable, fully responsive and works smoothly with all WordPress themes.

MotoPress Content Editor is an unparalleled website builder for non-tech people and a professional tool for designers.

== Installation ==

1. Upload the MotoPress plugin to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
You'll find 'MotoPress' in your Wordpress admin panel.

Read the [usage documentation](https://motopress.com/documentation/wordpress/plugins/) on the MotoPress website
